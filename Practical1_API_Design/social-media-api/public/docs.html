Social Media API Documentation

This is the documentation for the Social Media API, a RESTful API for a platform.

Users

GET /users

Get a list of all users with pagination

Query Parameters

ParameterTypeDescription
pageNumberPage number (default: 1)
limitNumberNumber of users per page (default: 10)

Response

{
  "success": true,
  "count": 2,
  "page": 1,
  "total_pages": 1,
  "data": [
    {
      "id": "1",
      "username": "traveler",
      "full_name": "Karma",
      "profile_picture": "https://example.com/profiles/traveler.jpg",
      "bio": "Travel photographer",
      "created_at": "2023-01-15"
    }
  ]
}
GET /users/:id

Get a specific user by ID

Parameters

ParameterTypeDescription
idStringUser ID

Response

{
  "success": true,
  "data": {
    "id": "1",
    "username": "traveler",
    "full_name": "Karma",
    "profile_picture": "https://example.com/profiles/traveler.jpg",
    "bio": "Travel photographer",
    "created_at": "2023-01-15"
  }
}
POST /users

Create a new user

Request Body

{
  "username": "new_traveler",
  "email": "new@example.com",
  "password": "securepassword",
  "full_name": "New Traveler",
  "bio": "Adventure seeker"
}

Response

{
  "success": true,
  "data": {
    "id": "4",
    "username": "new_traveler",
    "full_name": "New Traveler",
    "created_at": "2023-03-20"
  }
}
PUT /users/:id

Update an existing user

Request Body

{
  "full_name": "Updated Name",
  "bio": "Updated bio"
}

Response

{ "success": true, "data": { ...updatedUser } }
DELETE /users/:id

Delete a user

Response

{ "success": true, "data": {} }

Posts

GET /posts

Get a list of all posts with pagination

Response

{
  "success": true,
  "count": 3,
  "page": 1,
  "total_pages": 1,
  "data": [
    {
      "id": "1",
      "caption": "Beautiful sunrise!",
      "image": "https://example.com/posts/post1.jpg",
      "user_id": "1",
      "likes_count": 120,
      "user": {
        "id": "1",
        "username": "traveler",
        "full_name": "Karma",
        "profile_picture": "https://example.com/profiles/traveler.jpg"
      }
    }
  ]
}
GET /posts/:id

Get a specific post by ID

Parameters

ParameterTypeDescription
idStringPost ID
POST /posts

Create a new post (requires X-User-Id header)

Headers

HeaderDescription
X-User-IdThe ID of the authenticated user

Request Body

{
  "caption": "My new post",
  "image": "https://example.com/posts/newpost.jpg"
}
PUT /posts/:id

Update a post (requires X-User-Id header — must be post owner)

DELETE /posts/:id

Delete a post (requires X-User-Id header — must be post owner)

POST /comments

Request Body
